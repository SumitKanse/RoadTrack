# 🚧 Machine Learning Based Road Construction Monitoring System

## 📌 Overview
This project is a Machine Learning-based Image Processing System designed to monitor road construction progress using site images. It automates manual inspection by analyzing uploaded images and generating accurate progress reports.

The system improves efficiency, transparency, and decision-making in infrastructure projects.

---

## 🎯 Problem Statement
Traditional road construction monitoring:
- Requires frequent site visits  
- Is time-consuming and costly  
- Lacks real-time tracking  
- Leads to inconsistent reporting  

This project solves these issues using Machine Learning and Image Analysis.

---

## 💡 Key Features
- 📷 Upload construction site images  
- 🤖 ML-based stage detection (Earthwork, Paving, Finishing)  
- 📊 Automatic progress tracking & comparison  
- 📁 Centralized data storage  
- 👨‍💼 Role-based access (Admin, Contractor, Citizen)  
- 📈 Dashboard for visualization and reports  

---

## 🏗️ System Architecture
The system is divided into 4 layers:

1. Presentation Layer – React-based UI  
2. Application Layer – FastAPI backend  
3. ML Processing Layer – Image classification & analysis  
4. Data Layer – Database & storage

## ⚙️ How It Works
1. User uploads construction images  
2. System validates image quality  
3. ML model analyzes image  
4. Detects construction stage  
5. Compares with previous data  
6. Generates progress report  

---

## 👥 User Roles
- Admin – Approves users, monitors system  
- Contractor/Site Manager – Uploads images, updates projects  
- Citizen – Views progress and uploads comparison images
